动画- animation

我们来实现一个小男孩走动的动画。

                .walk {
                   /* 规定@keyframes 动画的名称 */
                   animation-name: person-walk;
                   /* 规定动画完成一个周期所花费的秒或毫秒。默认是0 */
                   animation-duration: 950ms;
                   /* 规定动画被播放的次数。默认是 1。infinite(循环播放) */
                   animation-iteration-count: infinite;
                   /* 动画切换的方式是一帧一帧的改变 */
                   animation-timing-function: step-start;
                }

                @keyframes person-walk {
                   0% {
                       background-position: -0px -291px;
                   }
                   25% {
                       background-position: -602px -0px;
                   }
                   50% {
                       background-position: -302px -291px;
                   }
                   75% {
                       background-position: -151px -291px;
                   }
                   100% {
                       background-position: -0px -291px;
                   }
                }

解释下定义的样式: 定义一个名为 person-walk 的@keyframes规则，@keyfrmaes用百分比来规定变化发生的时间, 0%是动画的开始，100%是动画的完成，规则中有4个百分比值的变化，每个比值分别 通过position获取一张图片，分别是0%，25%，50%，75%，100%。在950毫秒内从0%-100%发生4次变化， 采用的算法是 step-start 一帧一帧的切换，通过设置infinite参数每950毫秒不断循环。

技术点：css Sprites，即css精灵。 小男孩走动，靠不断的切换图片让人感觉视觉上不断在变化。 我们把小男孩走动的图片合成一张大图再利用css技术组合进行背景定位。 有 background-image、background-repeat、background-position。

小鸟动画的实现

在页面中增加一个小鸟的HTML结构。 通过background-position 加载精灵图，做动画的元素都是需要设置position: absolute 这样才能独立漂浮文档流，让页面的重绘更少。 图片变化部分使用 animation属性，通过设置animation-timing-function:step-start; 马上跳到动画每一帧结束的状态，这样就让动画执行一帧一帧的切换效果。

接下来我们结合下js技术，让小鸟飞到左边来。

我们引入了 jquery 和 jquery.transit 第三方插件，用来处理dom和动画效果。具体代码如下：

            var bird = {
                elem: $("#bird2"),
                fly: function(){
                    var me = this;
                    var width = $('.boxShow').width();
                    me.elem.addClass('birdFly').transition({
                        right: width
                    }, 15000, 'linear', function(){
                        me.elem.css({'right':0});
                        me.fly();
                    });
                }
            };

            bird.fly();

技术点：小鸟移动使用了css3的transition属性。
